### What’s happening here

**Axis shift**
- Floor1 (Balmora-like) = lateral civic grid  
- **Floor2 = longitudinal procession** → everything reinforces *forward movement*

**Sides have identity**
- **Top (trees)** = rooted, shaded, semi-sacred grove edge  
- **Bottom (horizon)** = exposed, bright, infinite read (water/void)

**Spine**
- `=` = wide promenade lanes (multi-lane NPC + player flow)
- Repeated `+` = access points, but less “decision,” more **rhythmic passing**

---

### Access Pattern

- **Tree side (`----+`)**
  - Entries feel **filtered / intentional**
  - Small kiosks, shrines, quiet nodes
  - NPC verbs: *linger, pray, observe, rest*

- **Horizon side (`####-##`)**
  - More **porous, commercial, active**
  - Docks, stalls, lookout points
  - NPC verbs: *trade, haul, depart, gather*

---

### NPC Flow (critical for “bustle”)

Think in **parallel lanes**:

- Inner lane (near trees): slow, contemplative traffic  
- Mid lane (center): main throughput (player default path)  
- Outer lane (horizon edge): fast movers, couriers, dock workers  

Cross-lane movement happens at each `+`, but:
- tree side = **low frequency crossings**
- horizon side = **high frequency crossings**

---

### Visual / Spatial Tricks

- Slightly **bias the spine downward** (closer to horizon) → constant peripheral view of openness
- Tree spacing irregular → breaks grid feel, adds **organic rhythm**
- Horizon edge can “breathe” (gaps, piers) → not a hard wall like previous floors
