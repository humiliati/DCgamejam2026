Overview
Per Tutorial_world_roadmap.md, COZY_INTERIORS_DESIGN.md, Biome Plan.html, CORE_GAME_LOOP_AND_JUICE.md, and TABLE_OF_CONTENTS_CROSS_ROADMAP.md, we need to unify and enhance how our floorN, floorN.N, and floorN.N.N levels interconnect and function as explorable, modular spaces.

Primary Tasks
Block out large, spacious floorN exteriors with tighter inner corridors that use shrubs/trees as guiding walls without making any of the map inaccessible. These corridors gently funnel players toward the "loudest" next floorN (the critical path), while allowing for a variety of distinct floorN.N entrances from the same floorN.
Encircle the playable space with a parameter (perimeter) wall of trees, then create a main arcade of buildings (shop, dungeon entrance, faction hideout, mechanic, utility, etc.) with shrubs as walls and walls of varying heights, fences of shrubs with special textures.
Each building should:
Have a generic exterior template (several shapes "L" "#" "T" type shapes where sensible /sizes per floorN).
Contain at least 3–5 buildings per floor for narrative structure (shop, hideout, mechanic, dungeon, etc).
Feature a distinct exterior for its floorN layer and corresponding unique interiors as floorN.Ns (use Gleaner's house exterior/interior as templates).
When applicable, connect to unique floorN.N.N subspaces, accentuating discovery doors.
Enhance modularity: New blockout files should be FRESH (not overwrite current ones); copy structure from existing, then "blow them up" for more space, embellish, and enrich.
Document any NEW textures needed for the texture atlas (attach list/spec for art pass).
Design Thinking
Brainstorm and sketch a set of generic building exterior shapes at multiple sizes for each floorN (e.g. rectangles, L-shapes, arcades, pavilions, etc).
Devise ways to make the floorN → floorN (critical route) visually/structurally the loudest path for new players, surrounded by but distinct from the many other optional building entrances.
Use shape, pathing, or environmental cues to highlight this progression.
For building interiors, use the Gleaner's house example to create a library of reusable templates — but make interior/exterior combos visually distinct for each building type (shop, faction hideout, etc).
Define how NPCs should be placed and how simple verb-pattern logic will allow sensible pathing around the expanded blockouts.
